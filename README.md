# WareHouse Of Crypto 💸💸

## 💻 Techs
- React.js
- Context API
- Axios
- React Icons







### ✅ Install Dependencies
With yarn:
```bash
cd ./crypto-app && yarn
```
With npm:
```bash
cd ./crypto-app && npm install
```
### 🏃‍♂️ Run the project
With yarn:
```bash
yarn start
```
With npm:
```
npm run start
```
